README.md
=========
DelegateBrowser delegates dedicated urls to different browsers

1. You must have checkout delegate-browser git hub project into C:\,
so current folder is C:\delegate-browser\windows7

2. open regedit, 
- import delegateBrowser.reg
- import http_userchoice_to_delegateBrowser.reg
- import https_userchoice_to_delegateBrowser.reg

3. configure C:\delegate-browser\IE-urlparts.txt :
- every line represents an URL part. Please enter a last empty line too
